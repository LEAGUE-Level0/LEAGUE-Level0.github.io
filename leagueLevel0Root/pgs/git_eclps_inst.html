Eclipse Github Plugin Guide

    
Cloning and Importing Code

    1. Get the URL for the repository you want to import and copy it.
    
    2. Go to File->Import
    
    3. Open the folder that reads "Git"
    4. Select "Projects from Git"
    
    5. Select "Clone URI"
    
   
    6. Paste the repoistory URL in the URI: box and click "Next >".
     
    
    7. From here, keep clicking "Next >" until the "Finish" box becomes available.
    
   
    
Committing and Pushing Code

   1. When you are ready to save your changes to GitHub, right click on the project folder in Eclipse.
   
   2. Select "Team" and then "Commit"
   
   3. Drag any items from the "Unstaged Changes" to the "Staged Changes" box.
   
   4. Add your commit message.
   
   5. Click "Commit and Push..." DO NOT CLICK "Commit"!!!
   
   6. If you accidentially clicked "Commit", right click on your project folder. 
	Then navigate to "Team" and then select "Push to Upstream". 
   
  
Troubleshooting

  1. Sometimes an error can occur when trying to clone a repository that already exists
	Simply navigate to that directory and delete the copy. 
	Retry the import and it should be successful.
